# Отправная точка

Пользователь активно использует Claude Code, но не осознавала что subagent-вызовы в сессии — это и есть агенты в действии. Концепция агента как «Claude с инструментами и несколькими шагами» была новой.

**Implications:** начинать с конкретных примеров из реального опыта, а не с абстрактных определений.
