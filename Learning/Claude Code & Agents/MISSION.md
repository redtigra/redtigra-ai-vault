# Mission: Claude Code & Agents

## Why
Я использую Claude Code для операционной работы и хочу перейти от реактивного использования к осознанному. Понять как устроены агенты — чтобы строить автоматизации самостоятельно, без разработчика.

## Success looks like
- Я понимаю что такое агент и чем он отличается от обычного запроса к Claude
- Я могу построить хотя бы одну небольшую агентную автоматизацию для реальной операционной задачи
- Я понимаю что происходит "под капотом" и могу направлять процесс — исправлять, уточнять, останавливать

## Constraints
- Не разработчик — объяснения без кода ради кода
- Работаю в Claude Code (не API, не кастомные приложения)
- Практика важнее теории — всё должно быть применимо к реальным задачам

## Out of scope
- Написание кода на Python/JS для продакшн-систем
- Деплой агентов в облако
- LLM-инфраструктура (fine-tuning, embedding, RAG)
