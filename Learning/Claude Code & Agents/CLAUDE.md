# Teaching Workspace: Claude Code & Agents

This is a stateful learning workspace. When the user runs `/teach`, read the following files to understand where we are:

- `MISSION.md` — what the user is trying to achieve
- `NOTES.md` — user preferences and context
- `learning-records/` — what has been covered so far
- `lessons/` — completed lessons (HTML)

The user is not a developer. Explain everything through real-world analogies and examples from their actual work (finance automation, Obsidian vault, operations).

Always start a session by reading the learning records to find the zone of proximal development before proposing the next lesson.
