Инструменты агента

← Урок 1: Что такое агент?

В прошлый раз мы разобрались: агент — это Claude с инструментами и несколькими шагами. Сегодня разберём что это за инструменты — и как ты остаёшься у руля.

Аналогия. Сам по себе Claude — это мозг без рук. Может думать, но ничего не сделает. Инструменты — это руки. Каждый инструмент даёт агенту одну конкретную возможность: прочитать, написать, запустить, найти.

Как работает агент на практике

Агент не делает всё за один шаг. Он идёт по циклу:

  1. Получает задачу от тебя
  2. Думает — какой инструмент нужен?
  3. Вызывает инструмент — получает результат
  4. Думает снова — что дальше?
  5. Повторяет до тех пор, пока задача не сделана

Это и есть агентный цикл. Инструменты — это то, что происходит на шаге 3.

Инструменты в Claude Code

Read — чтение

Открывает любой файл и читает его содержимое.

Пример: агент читает CSV с транзакциями перед обработкой.

Write / Edit — запись

Создаёт новый файл или изменяет существующий.

Пример: агент создаёт итоговый отчёт в Obsidian vault после анализа.

Bash / PowerShell — команды

Запускает команды в терминале — переименовывает файлы, запускает скрипты, перемещает папки.

Пример: агент перемещает обработанный счёт в папку /processed/.

Grep / Glob — поиск

Ищет файлы по имени или содержимому внутри проекта.

Пример: агент находит все файлы за март, чтобы сделать сводку.

WebFetch / WebSearch — интернет

Загружает страницу по ссылке или делает поиск в сети.

Пример: агент claude-code-guide читал документацию из интернета в прошлый раз.

Agent — субагент

Запускает другого Claude как отдельного агента с чистым контекстом.

Пример: главный агент поручает claude-code-guide найти документацию, пока сам занимается другим.

Ты остаёшься у руля

Агент не может просто делать что хочет. Прежде чем использовать некоторые инструменты — особенно те, которые что-то меняют — Claude Code спрашивает разрешения.

Вот как это выглядит в терминале:

Claude wants to run a command:
PowerShell: Move-Item invoice_march.pdf ./processed/

[y] Allow   [n] Deny

Ты видишь точно что агент собирается сделать — и можешь сказать нет. Если скажешь нет — агент это учитывает и думает дальше (или останавливается).

Аналогия. Ты дала коллеге задание. Но прежде чем он отправит письмо клиенту — он показывает тебе черновик. Ты читаешь, правишь, говоришь «окей». Агент работает так же: инициатива у него, решение у тебя.
